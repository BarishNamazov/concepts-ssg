# Hello, World!

This is a static site generated with **concept-design** — a framework where independent, reusable concepts are composed by declarative synchronizations.

## How It Works

1. **Filing** discovers markdown files
2. **Frontmattering** parses metadata
3. **Formatting** converts markdown to HTML
4. **Routing** derives clean URL paths
5. **Layouting** composes pages from nestable layouts

Check out the [blog](/blog) for more posts.
